Uso semântico de listas

Elementos de lista devem ser utilizados para agrupar itens que compartilham uma mesma natureza ou são partes de um conjunto. Por exemplo, uma lista de recursos, uma lista de passos em um procedimento, uma lista de itens de menu, etc.

O uso de elementos HTML semânticos apropriados para listas é fundamental. O elemento <ul> (lista não ordenada) é usado para itens sem uma ordem específica, enquanto <ol> (lista ordenada) é utilizado quando a sequência dos itens é importante. Os itens individuais são definidos com <li> (item de lista).

Deve-se evitar a utilização de números para identificar uma lista não ordenada, já que isso contradiz a semântica programática do conteúdo. Para esses casos, é recomendável utilizar o elemento semântico correto (<ol>). Também não é uma boa prática criar uma lista onde cada item é outra lista de apenas um item.

Listas bem estruturadas ajudam leitores de tela a comunicar a relação entre itens, facilitando a navegação e compreensão do conteúdo. Isso é importante para garantir que todos os usuários possam entender o agrupamento e a ordem dos itens listados.

Exemplo de lista não ordenada

O exemplo a seguir traz uma lista não ordenada com algumas possíveis tarefas diárias de um product owner. Uma lista não ordenada (<ul>) foi usada para agrupar itens que não precisam de uma ordem específica. Cada item da lista foi especificado com o uso de <li>, mantendo a estrutura clara e semanticamente compreensível.

  • Revisar backlog do produto
  • Priorizar itens para o próximo sprint
  • Alinhar requisitos com stakeholders
  • Participar da reunião diária (daily meeting)
  • Especificar novas histórias de usuário
  • Verificar progresso das tarefas em andamento
  • Revisar feedback de clientes
  • Planejar a revisão do sprint

Exemplo de lista ordenada

Neste exemplo uma lista ordenada é utilizada para elencar os passos na ordem correta para restaurar um dispositivo Android para as configurações de fábrica. Uma lista ordenada (<ol>) foi usada para passos ou itens que precisam seguir uma sequência específica. Cada item da lista foi especificado com o uso de <li>, mantendo a estrutura clara e semanticamente compreensível.

  1. Fazer backup dos seus dados importantes (contatos, fotos, aplicativos)
  2. Acessar as "Configurações" do dispositivo
  3. Descer até a seção "Sistema" e tocar em "Redefinir"
  4. Selecionar "Apagar todos os dados (restauração de fábrica)"
  5. Ler as informações e confirmar a ação
  6. Inserir o PIN, senha ou padrão, se solicitado
  7. Tocar em "Apagar todos os dados" para iniciar a restauração
  8. Aguardar o processo de reinicialização e restauração
  9. Configurar o dispositivo novamente após a reinicialização
  10. Restaurar os dados do backup, se necessário
