Critério 1.4.5

Imagens

5. Imagens de Texto: Não há imagens de texto ou todas as imagens de texto são essenciais e possuem texto alternativo com o mesmo conteúdo.

De acordo com o critério 1.4.5 das Diretrizes de Acessibilidade para Conteúdo Web (WCAG), deve-se evitar o uso de imagens que contenham texto, a menos que essas imagens sejam consideradas essenciais e não possam ser representadas de outra forma. Se for necessário o uso de uma imagem de texto, ela deve conter uma descrição alternativa (atributo alt) com o mesmo conteúdo textual, de forma que o usuário possa compreender a informação mesmo que não consiga visualizar a imagem.

O exemplo abaixo simula uma imagem de texto usada como banner promocional e faz parte do design visual específico da página de e-commerce demonstrada. A imagem possui um texto alternativo que descreve seu conteúdo, garantindo acessibilidade para pessoas que usam leitores de tela.

Explicação do Exemplo:

Neste exemplo, a imagem simula um banner promocional com um texto que é parte integral do design visual e por isso é essencial. O atributo alt contém o mesmo conteúdo textual da imagem, permitindo que usuários de leitores de tela recebam a mesma informação. É importante lembrar que, sempre que possível, o conteúdo textual deve ser representado em HTML para facilitar o acesso, redimensionamento e personalização pelo usuário.
