Critério 3.1.3

Conteúdo textual

9. Uso de texto especial: Todos os trechos de texto de ênfase, citação, abreviação ou texto especial são utilizados para transmitir esses propósitos.

O critério 3.1.2 das WCAG recomenda que trechos de texto com propósitos específicos, como ênfase, citações ou abreviações, sejam corretamente marcados com elementos semânticos específicos. Isso melhora a acessibilidade, garantindo que tecnologias assistivas transmitam a informação corretamente.

Utilizar a marcação semântica apropriada facilita a compreensão do conteúdo para usuários de leitores de tela, além de fornecer contexto adicional sobre a função e o significado de partes específicas do texto.

O uso correto de elementos semânticos oferece os seguintes benefícios:

  • Facilita a compreensão do conteúdo por usuários de tecnologias assistivas.
  • Melhora a estrutura e a legibilidade do conteúdo.
  • Fornece contexto adicional para termos ou frases específicas.

O exemplo a seguir demonstra o uso correto de elementos semânticos para ênfase, citações e abreviações:

Explicação do Exemplo:

  • Ênfase: O elemento <em> é utilizado para dar ênfase ao texto “destacar”. Leitores de tela geralmente modificam o tom de voz para transmitir essa ênfase.
  • Citação: O elemento <cite> é usado para identificar a fonte de uma citação, no caso, “Albert Einstein”.
  • Bloco de Citação: O elemento <blockquote> é utilizado para destacar uma citação longa, indicando que o texto foi citado de outra fonte.
  • Abreviação: O elemento <abbr> é utilizado para definir a sigla “WWW” (World Wide Web). O atributo title fornece a expansão completa da sigla, ajudando leitores de tela a transmitir o significado completo.
  • Realce: Uso de <strong>: Realça visualmente e semanticamente um termo importante como "acessibilidade".

É importante destacar certas informações. Como disse Albert Einstein:

"A imaginação é mais importante que o conhecimento."

Siglas como WWW são amplamente conhecidas.

O termo acessibilidade deve ser priorizado em qualquer projeto web.

Em casos de abreviações, como CSS, é importante fornecer o significado completo.

Para dar ênfase em uma palavra ou frase, usamos a tag em. Por exemplo, inclusão é um conceito fundamental.

Palavras técnicas, como aria-label, devem ser explicadas em seu contexto.

Se precisar referenciar um livro ou trabalho acadêmico, utilize a tag <cite>. Por exemplo, em Design para Todos, o autor destaca a importância da acessibilidade.
