Critério 3.1.5

Conteúdo Textual

12. Nível de linguagem: Todo conteúdo de texto utiliza linguagem simples e clara ou existe conteúdo suplementar ou uma versão alternativa simplificada para linguagem especializada ou recursos linguísticos complexos.

O critério 3.1.5 das WCAG recomenda que o conteúdo textual utilize linguagem simples e clara. Caso o conteúdo seja especializado ou complexo, deve-se fornecer uma explicação suplementar ou uma versão alternativa simplificada.

Textos com linguagem simples facilitam a compreensão para pessoas com deficiência cognitiva, dificuldades de leitura ou para quem não domina completamente o idioma. Conteúdos complexos ou técnicos devem ser complementados com explicações acessíveis.

O uso de linguagem simples e clara proporciona os seguintes benefícios:

  • Facilita a compreensão para usuários com deficiência cognitiva.
  • Melhora a experiência de pessoas com baixo letramento ou não nativas no idioma.
  • Promove a inclusão, garantindo que mais usuários possam entender o conteúdo.

A Otimização de Sites, também conhecida como SEO, é um conjunto de técnicas para melhorar a visibilidade de um site nos mecanismos de busca.

Explicação Simplificada:

SEO é um conjunto de estratégias que ajuda um site a aparecer melhor em buscas como o Google. Por exemplo, quando você procura algo, o SEO ajuda a mostrar os sites mais relevantes primeiro.
