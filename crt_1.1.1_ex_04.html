Alternativa textual para mapa de imagem

Deve ser fornecida uma alternativa textual para a imagem como um todo, utilizando o atributo alt da imagem em questão e descrições para cada uma das áreas ativas, ou seja, para cada das áreas mapeadas na imagem pois elas se comportam como links que recebem o foco. Para descrever cada uma das áreas também utiliza-se o atributo alt.

Mapa de imagem É um elemento HTML que permite que diferentes áreas de uma imagem sejam clicáveis. Ou seja, consiste em uma lista de coordenadas de uma imagem, que são links para vários destinos.

Exemplo

A imagem do sistema solar, a seguir, permite que o usuário acesse mais informações sobre cada um dos planetas através de links. O texto alternativo para a imagem como um todo, neste caso, foi descrito como "ilustração do sistema solar com a representação das órbitas dos planetas". Cada um dos planetas e o sol foram mapeados e para cada um deles foi inserido no atributo alt da tag area o seu respectivo nome.
