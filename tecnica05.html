Alternativa textual para imagens decorativas

Se o conteúdo não textual for meramente decorativo, se for utilizado apenas para formatação visual, ou se não for exibido aos usuários, então esse conteúdo não textual deve ser implementado de uma forma que possa ser ignorado pelas tecnologias assistivas. Isso pode ser feito de duas principais formas:

1 - Usando CSS para incluir imagens decorativas.

Essa técnica tem por objetivo fornecer um mecanismo que permita adicionar imagens decorativas ou utilizadas para formatação visual no conteúdo Web, sem exigir marcação adicional no conteúdo. Dessa forma, as tecnologias assistivas conseguem ignorar completamente esse conteúdo não textual.

Imagens de fundo podem ser incluídas com as seguintes propriedades CSS:

  • background;
  • background-image;
  • content, combinado com os pseudo-elementos “::before” e “::after”;
  • list-style-image.

Exemplo de uma imagem de fundo inserida via CSS

Exemplo de código:

body { background: #ffe url('/imagens/background.jpg') repeat; }

Exemplo de uma imagem inserida via CSS para efeito de hover do mouse sobre um link

Exemplo de código:

a:hover { background: #ffe url('/imagens/flutuar.jpg') repeat; color: #000; text-decoration: none; }

2 - Deixando o atributo “alt” nulo e sem atributo “title” em imagens que a tecnologia assistiva deva ignorar.

O objetivo desta técnica é mostrar como imagens decorativas podem ser inseridas de forma que sejam ignoradas pela tecnologia assistiva.

Se não for utilizado o atributo “title”, e o atributo “alt” estiver nulo (exemplo: alt = "" - sem espaço entre as aspas) os softwares de tecnologia assistiva saberão que aquela imagem pode ser ignorada com segurança e sem prejuízo para a compreensão do conteúdo da página.

Exemplo de uma imagem decorativa em um página web:

Exemplo de código:

<img alt="" src="item.jpg" height="25" width="25">
